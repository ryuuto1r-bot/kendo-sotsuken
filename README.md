Kendo Virtual Coach
===================

剣道の自主練で使う、素振り・打突動作の相対評価ツールです。

GitHubで共有・管理する場合の手順は `GITHUB_SETUP.md` にまとめています。
他の人にURLで使ってもらう本番公開は、HTTPSとカメラ/WASM向けヘッダーを扱いやすいCloudflare PagesまたはNetlifyを推奨します。

公開用の卒研説明ページは `research/` にまとめています。
Codex引き継ぎ用の内部メモはローカル専用として管理し、公開リポジトリには含めません。

採用方針
--------

- 本採用: `index.html`
  - MediaPipe Pose版です。
  - 相対評価、角度、軸、右手首基準の打突時間を安定して見る用途に使います。
  - 2026-05-18時点で、初期の安定版 ver.9 をベースに戻し、起動に必要な最小修正だけを入れています。
  - requestVideoFrameCallback対応、3D速度推定、信頼度連動フィルタ、Savitzky-Golay後処理、竹刀先端テープROI探索を追加しています。
  - MediaPipe Tasks Pose Landmarkerを優先し、ローカル同梱の `vision_bundle_browser.js` で読み込みを安定化しています。ブラウザ側で読み込めない場合は従来Poseへ自動フォールバックします。
- 比較版: `yolo.html`
  - YOLOv8 Pose + ONNX Runtime Web版です。
  - 研究・比較・将来検証用として残しています。

使い方
------

`file://` で直接開くとカメラやAI処理が不安定です。必ずローカルサーバーで開いてください。

Pose Landmarkerを安定させたい場合はこちらを使います。

```sh
cd /Users/user/Documents/Codex/2026-04-27/files-mentioned-by-the-user-4211725
python3 serve-kendo.py
```

その後、ブラウザで `http://127.0.0.1:4174/index.html` を開きます。
このサーバーはMediaPipe Tasks / WASM向けのヘッダーを返すため、`python3 -m http.server` より安定します。

簡易確認だけなら通常サーバーでも開けます。

```sh
cd /Users/user/Documents/Codex/2026-04-27/files-mentioned-by-the-user-4211725
python3 -m http.server 4174 --bind 127.0.0.1
```

その後、ブラウザで開きます。

- MediaPipe本採用: `http://127.0.0.1:4174/index.html`
- YOLO比較版: `http://127.0.0.1:4174/yolo.html`

`.command` から開く場合
----------------------

- `start-kendo-camera.command`: MediaPipe本採用版を開きます。
- `start-yolo-mirror.command`: YOLO比較版を開きます。

先生への説明
------------

現在は、リアルタイムの相対評価ではMediaPipe版を本採用しています。
理由は、YOLO版より処理が軽く、フレーム落ちが少ないため、打突時間や角度変化のような時間変化を安定して比較しやすいからです。

YOLO版は1フレームの姿勢推定精度を検証する比較用として残しています。
ただし、ブラウザ上でのリアルタイム解析では推論負荷が高く、打突のような一瞬の動作ではフレーム落ちが計測誤差につながりやすいです。

計測上の注意
------------

このツールは接触式測定器ではなく、カメラ映像からの推定です。
厳密な竹刀先端速度を測るには、高fps動画解析、竹刀先端の色テープ検出、毎回同じカメラ距離・角度での校正が必要です。
速度表示の `m/s*` は肩幅から換算した推定値、`m/s` はMediaPipeの3Dワールド座標を利用できた場合の値です。

今回追加した操作
--------------

- 動画解析: フレーム送り、開始/終了フレームの手動補正、骨格つきプレビュー、CSVレポート出力
- 履歴: 1件ごとのメモ、削除、クリップHTML保存
- カメラ: 内蔵/外部カメラ選択、カメラ解像度と実測fps表示
- 練習画面: 計測タブをほぼ全画面化し、練習後は打突詳細タブで速度・角度を確認

打突時間の定義
--------------

- 開始: 右手首の下方向速度が閾値を超えた最初のフレーム
- 終了: 最高速度後、速度が急減速したフレーム

この定義は画面内の「打突時間の定義」にも表示しています。
