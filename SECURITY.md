# Security / Privacy

このアプリは、剣道の素振り・打突動作をブラウザ上で解析する研究用ツールです。

## 動画・カメラデータ

- カメラ映像と動画解析は、基本的に利用者のブラウザ内で処理します。
- 現在の静的Web版では、動画ファイルをサーバーへ自動送信する処理は入れていません。
- 履歴データはブラウザのlocalStorageに保存されます。

## 利用時の注意

- 公開URLでカメラを使うにはHTTPSが必要です。
- 共有PCや学校PCで使う場合、使用後に履歴を削除してください。
- 個人が映る動画を第三者に共有する場合は、本人の許可を取ってください。

## 測定値について

このアプリの速度・角度・打突時間は、カメラ映像と姿勢推定に基づく推定値です。
接触式測定器や公式計測器の代替ではありません。
